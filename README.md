# Customer-Churn-Prediction
Customer Churn Prediction using Machine Learning techniques to analyze customer behavior and predict whether a customer is likely to leave a company. Includes data preprocessing, exploratory data analysis (EDA), model training, evaluation metrics, and visualization using Python libraries such as Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn.

# 📊 Customer Churn Analytics & Prediction Dashboard

## 📌 Project Overview

This project focuses on analyzing customer churn patterns and identifying key factors that influence customer retention. Using Python for data preprocessing and machine learning, and Power BI for visualization, the project provides actionable business insights to help organizations reduce customer churn.

---

## 🎯 Objectives

* Analyze customer churn behavior.
* Identify key factors contributing to churn.
* Visualize customer retention trends.
* Generate business insights for customer retention strategies.
* Build an interactive Power BI dashboard for decision-making.

---

## 📂 Dataset

**Dataset:** IBM Telco Customer Churn Dataset

The dataset contains customer demographics, account information, services subscribed, monthly charges, tenure, and churn status.

## 💡 Key Insights

* Overall churn rate is 26.54%.
* Month-to-Month contracts have the highest churn rate.
* Fiber Optic customers show higher churn risk.
* Electronic Check users contribute significantly to churn.
* Customers with higher monthly charges are more likely to churn.
* Long-term contracts improve customer retention.
* Senior citizens exhibit higher churn tendencies.

---

## 🎯 Business Recommendations

* Encourage customers to switch to long-term contracts.
* Offer personalized retention programs for high-risk customers.
* Improve customer engagement during the initial months of service.
* Review pricing strategies for high monthly charge customers.
* Enhance customer support for Fiber Optic service users.
